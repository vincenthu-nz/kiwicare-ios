# KiwiCare iOS

KiwiCare is an on-demand home services app built with SwiftUI.

![CI](https://github.com/vincenthu-nz/KiwiCare-iOS/actions/workflows/ios-ci.yml/badge.svg)

## Requirements

- Xcode 16+
- iOS 17+
- Swift 6

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/vincenthu-nz/KiwiCare-iOS
   ```
2. Open `KiwiCare/KiwiCare.xcodeproj` in Xcode
3. Select a simulator and press **Run**

## CI

GitHub Actions runs on every push and pull request to `main`:
- Build (Release, no code signing)
- Unit Tests
